# F-14B Tomcat (RIO seat)

Displays the RIO's Computer Address Panel (CAP) — the tactical data entry panel used by the
Radar Intercept Officer. The CDU shows the active HCU mode, WCS radar mode, and the ten
label lines of the currently selected CAP category page.

## Supported devices

| Device | What it shows |
|--------|---------------|
| CDU (MCDU / PFP3N / PFP7 / PFP4) | CAP category pages (HCU mode, WCS mode, 5 label pairs) |

## CDU display

The display is organised as follows:

| Line | Content | Colour |
|------|---------|--------|
| 0 | `HCU-mode   WCS-mode   CAP-category` | White |
| 1 | *(blank)* | — |
| 2–3 | CAP category label pair 1 | White / Cyan |
| 4–5 | CAP category label pair 2 | White / Cyan |
| 6–7 | CAP category label pair 3 | White / Cyan |
| 8–9 | CAP category label pair 4 | White / Cyan |
| 10–11 | CAP category label pair 5 | White / Cyan |

Each label pair uses an alternating colour scheme (white upper line, cyan lower line) to match
the CAP panel layout in the cockpit.

### HCU modes

The Hand Control Unit (`RIO_HCU_*`) selects what the TID (Tactical Information Display) is slaved to:

| Value | Label |
|-------|-------|
| TID | TID display |
| TCS | Television Camera Set |
| RADAR | Radar |
| DDD | Detail Data Display |

### WCS radar modes

The Weapon Control System (`RIO_RADAR_*`) indicates the current radar search/track mode:

| Value | Label |
|-------|-------|
| TWS Auto | Track While Scan — automatic |
| Puls Sch | Pulse search |
| TWS Man | Track While Scan — manual |
| PD Sch | Pulse Doppler search |
| PD STT | Pulse Doppler Single Target Track |
| RWS | Range While Search |
| Puls Stt | Pulse Single Target Track |

### CAP categories (`RIO_CAP_CATRGORY`)

| Index | Category |
|-------|----------|
| 0 | BIT — Built-In Test |
| 1 | SPL — Special |
| 2 | NAV — Navigation |
| 3 | TAC DATA — Tactical Data |
| 4 | D/L — Data Link |
| 5 | TGT DATA — Target Data |

## Brightness / lighting

No dedicated brightness control.

## Front panels

None — FCU/EFIS, PAP3, and AGP32 panels stay idle.

## Notes

- The F-14 pilot has no traditional CDU; the CAP is exclusively a RIO instrument.
  The bridge therefore shows the RIO CAP data regardless of which seat is flying.
- `RIO_CAP_CATRGORY` is the DCS-BIOS control identifier — the typo is in the DCS-BIOS JSON
  and must be preserved exactly.
- Covers both the F-14B and F-14A-135-GR variants (detected automatically).
