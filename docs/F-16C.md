# F-16C Viper

Basic support: a switchable CDU showing the DED, a NAV page, or the RWR. The console
brightness is shared with front panels for lighting only.

## Supported devices

| Device | What it shows |
|--------|---------------|
| CDU (MCDU / PFP3N / PFP7 / PFP4) | DED / NAV / RWR pages, Master Caution LED |
| FCU + EFIS / PAP3 | Brightness only (no flight data is published) |
| AGP32 | Gear position lights + warning, brightness |

## CDU display

The CDU shows one of three pages at a time; switch with the configured keys:

- **DED** (default) — the Data Entry Display.
- **NAV** — navigation page.
- **RWR** — radar-warning receiver page.

Defaults: **PrevPage** -> DED, **NextPage** -> NAV, **LineSelectRight1** -> RWR. All three
keys are configurable in the options.

### LED mapping

| CDU LED | DCS indicator |
|---------|---------------|
| FAIL | Master Caution |

## Brightness / lighting

- **Display brightness** follows the data-display brightness knob.
- **Backlight brightness** follows the primary consoles brightness knob.
- The console brightness is also forwarded to front panels so they dim with the cockpit.
- Use the global **"Disable lighting management"** option to opt out.

## AGP32

The AGP32 gear panel receives live gear-position and warning data from DCS-BIOS:

| AGP32 LED | DCS indicator |
|-----------|---------------|
| Gear 1 (left) green | Left main gear down-and-locked (`LIGHT_GEAR_L`) |
| Gear 2 (nose) green | Nose gear down-and-locked (`LIGHT_GEAR_N`) |
| Gear 3 (right) green | Right main gear down-and-locked (`LIGHT_GEAR_R`) |
| UNLK triangles (all three) + red arrow | Gear-in-transit / warning (`LIGHT_GEAR_WARN`) |

The gear lights remain visible even when cockpit console brightness is at zero.

## Front panels

FCU/EFIS and PAP3 only receive **console brightness** — no speed/heading/altitude is
published — so they dim with the cockpit but stay otherwise idle.

## Notes

- Display-mode keys can be remapped via the `F16CPrevDisplayKey`, `F16CNextDisplayKey`, and
  `F16CRwrDisplayKey` options.
