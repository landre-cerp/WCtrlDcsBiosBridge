# M-2000C

Basic support: the PCN (navigation control/display) and a summarized caution-light panel
are shown on the CDU, with PCN lights mapped to the CDU LEDs. The AGP32 gear panel
receives live gear-position and warning data.

## Supported devices

| Device | What it shows |
|--------|---------------|
| CDU (MCDU / PFP3N / PFP7 / PFP4) | PCN coordinates/PREP/DEST, a condensed caution list, PCN status LEDs |
| AGP32 | Gear position lights + warning |

## CDU display

- **PCN display** — the left/right coordinate windows plus the PREP and DEST numbers.
- **Caution panel** — the cockpit Caution Light Panel (CLP) is decoded from its bitfield
  registers and listed on the CDU, colour-coded by severity (advisory / warning / critical).

### LED mapping

The PCN lights register drives the CDU LEDs:

| CDU LED | PCN light |
|---------|-----------|
| FM1 | ALN (alignment) |
| RDY | PRET (ready) |
| FM | NDEG |
| IND | MIP |
| FM2 | SEC |
| FAIL | UNI |

## Brightness / lighting

- No dedicated brightness control.

## AGP32

The AGP32 gear panel receives live gear-position and warning data from DCS-BIOS:

| AGP32 LED | DCS indicator |
|-----------|---------------|
| Gear 1 (left) green | Left main gear down-and-locked (`CONF_G`) |
| Gear 2 (nose) green | Aux gear down-and-locked (`CONF_AUX`) |
| Gear 3 (right) green | Right main gear down-and-locked (`CONF_D`) |
| UNLK triangles (all three) + red arrow | CONF panel red warning (`CONF_ROUGE`) and gear lever light (`LANDING_GEAR_LEVER_LIGHT`) |

## Front panels

FCU/EFIS and PAP3 stay idle — the M-2000C does not publish speed, heading, or altitude.

## Notes

- The light/CLP registers are read from raw DCS-BIOS addresses as a workaround for a
  module export mismatch; if the upstream definitions are fixed this may change.
