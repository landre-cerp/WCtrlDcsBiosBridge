# F-15E Strike Eagle

Basic support: the six Up-Front Control (UFC) display lines are mirrored to the CDU.
No front panel feed.

## Supported devices

| Device | What it shows |
|--------|---------------|
| CDU (MCDU / PFP3N / PFP7 / PFP4) | UFC display lines 1-6, centred on screen |

## CDU display

The six UFC lines (`F_UFC_LINE1`–`F_UFC_LINE6`) are shown centred, with colours roughly
matching the cockpit display (white outer lines, red middle lines).

## Brightness / lighting

- No dedicated brightness control.

## Front panels

None — the F-15E does not publish flight-deck data, so FCU/EFIS, PAP3, and AGP32 panels
stay idle.

## Notes

- UFC display contributed by **smreki**.
